# Netflix Data Analysis

## Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python and Pandas. It explores trends in content types, ratings, release years, countries, and genres through data cleaning and visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
Netflix Movies and TV Shows Dataset

## Analysis Performed
- Checked dataset information
- Checked missing values
- Cleaned missing data
- Counted Movies vs TV Shows
- Top 10 content-producing countries
- Content rating distribution
- Content added over the years
- Top 10 genres on Netflix
- Release year analysis

## Visualizations
- Bar Charts
- Count Plots
- Line Charts

## Conclusion
This analysis provides insights into Netflix's content library, showing trends in content types, ratings, genres, and country-wise production using Python and Pandas.
